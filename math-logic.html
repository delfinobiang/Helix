Every decision your computer makes — every if, every while, every filter, every search — comes down to one question: TRUE or FALSE? That's Boolean logic, and it's the heartbeat of everything digital.

1
What is Boolean Logic?
In 1854, a mathematician named George Boole published a system of algebra where every variable had only two possible values: TRUE or FALSE. He called it "The Laws of Thought."

Nearly 100 years later, engineers realised that electrical circuits — which are either ON or OFF — could represent Boole's algebra perfectly. Every transistor in your computer is a Boolean switch. Your phone has roughly 15 billion of them.
💡
Boolean in Code
In Python, True and False are actual values. In JavaScript, any value can be "truthy" or "falsy". In every language, every condition evaluates to one of these two states.
python — boolean values in action
# Boolean values is_logged_in = True has_permission = False # Every comparison returns a boolean score = 85 passed = score >= 60 # True failed = score < 60 # False print(passed) # True

The AND Gate — Both must be true
The AND operator returns True only when both inputs are true. Think of it like a door that needs two keys: unless both are inserted, the door stays locked.

In code you write and (Python) or && (JavaScript/C++).

OR and NOT Gates
OR returns True if at least one input is true. Like a door that opens with key A or key B — either one works.

NOT flips the value. True becomes False, and False becomes True. It only takes one input.

python — OR and NOT in real code
is_admin = False is_owner = True # OR: either condition is enough can_delete = is_admin or is_owner print(can_delete) # True # NOT: flip the value is_guest = not is_admin print(is_guest) # True # Combine them is_blocked = False can_view = (is_admin or is_owner) and not is_blocked print(can_view) # True

XOR & Building a Truth Table
XOR (Exclusive OR) returns True only when the inputs are different. If both are the same — both TRUE or both FALSE — it returns False.

XOR is used everywhere: encryption, error detection, toggle buttons in UI, even flipping bits. Now let's put everything together and fill in a truth table yourself.
