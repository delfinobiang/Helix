Lesson 01 · Data & AI

What is Data?

Data is raw recorded observations about the world. By itself it's meaningless — it becomes valuable through processing, analysis, and interpretation. Understanding what data is and isn't is the foundation of data science.

📊 Data vs Information vs Knowledge
  • Data: Raw facts — "23, 45, 12, 67"
  • Information: Data in context — "User ages: 23, 45, 12, 67"
  • Knowledge: Information interpreted — "Most users are 20–50 years old"
Data TypeDescriptionExamples
StructuredRows and columns with defined schemaSQL tables, CSV files
UnstructuredNo fixed formatText, images, audio, video
Semi-structuredSome structure but flexibleJSON, XML, logs
Time seriesData indexed by timeStock prices, sensor readings
SpatialLocation-based dataGPS coordinates, maps

Lesson 02 · Data & AI

Data Types & Formats

Data comes in different statistical types that determine which operations make sense. Confusing types leads to incorrect analysis — like calculating the average of zip codes.

📐 Four Levels of Measurement
  • Nominal: Categories with no order — colours, genders, countries
  • Ordinal: Ordered categories — star ratings (1–5), education level
  • Interval: Equal gaps, no true zero — temperature (°C), calendar years
  • Ratio: Equal gaps + true zero — height, weight, age, income

Lesson 03 · Data & AI

Cleaning & Wrangling

Real-world data is messy. Data scientists spend 60–80% of their time cleaning data before any analysis. Learning to handle missing values, duplicates, and inconsistencies is essential.

⚠️ Common Data Quality Issues
  • Missing values: NaN, null, empty strings — can't be ignored
  • Duplicates: Same record appearing multiple times
  • Inconsistent formats: "2024-01-15" vs "15/01/2024" vs "Jan 15"
  • Outliers: Values far outside the normal range
  • Wrong types: Numbers stored as strings
import pandas as pd df = pd.read_csv('users.csv') # Check for missing values print(df.isnull().sum()) # Drop rows with any NaN df.dropna(inplace=True) # Fill missing with column mean df['age'].fillna(df['age'].mean(), inplace=True) # Remove duplicates df.drop_duplicates(inplace=True) # Fix types df['age'] = df['age'].astype(int)

Lesson 04 · Data & AI

Exploratory Data Analysis

EDA is the process of summarising, visualising, and understanding a dataset before modelling. It reveals patterns, anomalies, and relationships that guide all further analysis.

🔍 EDA Checklist
  • Shape: how many rows and columns?
  • Types: what is each column's data type?
  • Missing: which columns have nulls?
  • Distribution: what's the spread of each numeric column?
  • Correlations: which columns move together?
  • Outliers: any extreme values that need attention?
import pandas as pd df = pd.read_csv('data.csv') df.shape # (rows, columns) df.info() # types + non-null counts df.describe() # mean, std, min, max, quartiles df.value_counts() # frequency of each category df.corr() # correlation matrix # Quick visualisation import matplotlib.pyplot as plt df['age'].hist(bins=20) plt.show()
