Lesson 01 · Data & AI

The Perceptron

A perceptron is the simplest neural unit — it takes inputs, multiplies each by a weight, adds a bias, and passes the result through an activation function. It's a mathematical model of a biological neuron.

🧠 Perceptron Formula

output = activation(w₁x₁ + w₂x₂ + ... + wₙxₙ + b)

Where: x = inputs, w = weights (learned), b = bias (learned), activation = non-linear function (ReLU, sigmoid)

⚡ Common Activation Functions
  • ReLU: f(x) = max(0, x) — most used in hidden layers
  • Sigmoid: f(x) = 1/(1+e⁻ˣ) → (0,1) — for binary output
  • Softmax: converts outputs to probabilities summing to 1 — multi-class
  • Tanh: f(x) → (−1, 1) — zero-centered

Lesson 02 · Data & AI

Multi-Layer Networks

Stack perceptrons into layers and you get a multi-layer perceptron (MLP). Each layer learns increasingly abstract features. The first layer detects edges; the last layer detects concepts.

🏗️ Network Architecture
  • Input layer: One neuron per feature (e.g. 784 for a 28×28 image)
  • Hidden layers: Learn intermediate representations (most of the magic)
  • Output layer: One neuron per class (or 1 for regression/binary)
import torch import torch.nn as nn class MLP(nn.Module): def __init__(self): super().__init__() self.layers = nn.Sequential( nn.Linear(784, 256), # input → hidden 1 nn.ReLU(), nn.Linear(256, 128), # hidden 1 → hidden 2 nn.ReLU(), nn.Linear(128, 10), # hidden 2 → output (10 classes) ) def forward(self, x): return self.layers(x)

Training & Backpropagation

Backpropagation is the algorithm that trains neural networks. It computes how much each weight contributed to the error, then nudges all weights in the direction that reduces error.

🔄 Training Loop
  • Forward pass: Input → layers → prediction
  • Loss: Compare prediction to truth (e.g. cross-entropy loss)
  • Backward pass: Compute gradients via chain rule (backprop)
  • Update: w = w − learning_rate × gradient
  • Repeat: Thousands of times over the dataset
# PyTorch training loop optimizer = torch.optim.Adam(model.parameters(), lr=1e-3) criterion = nn.CrossEntropyLoss() for epoch in range(10): for X_batch, y_batch in dataloader: optimizer.zero_grad() # reset gradients output = model(X_batch) # forward pass loss = criterion(output, y_batch) # compute loss loss.backward() # backprop optimizer.step() # update weights

Gradient Descent Direction

If a weight's gradient is positive (increasing the weight increases loss), should you increase or decrease the weight? Type: decrease

Gradient descent: w = w − lr × gradient. Positive gradient → subtract → decrease weight → reduce loss.
Lesson 04 · Data & AI

Deep Learning

Deep learning uses neural networks with many layers. CNNs revolutionised image recognition. Transformers power GPT and BERT. These architectures are what makes modern AI possible.

🏛️ Key Architectures
  • CNN (Convolutional): Images — detects patterns at different scales
  • RNN/LSTM: Sequences — text, audio, time series
  • Transformer: Attention mechanism — GPT, BERT, LLMs
  • GAN: Generates realistic synthetic data
  • Diffusion Model: DALL·E, Stable Diffusion image generation
👁️
Computer Vision
CNNs power face recognition, medical imaging, autonomous vehicles, and photo organisation.
💬
Language Models
GPT, Claude, Gemini — transformers trained on trillions of tokens power modern AI assistants.
🎨
Image Generation
DALL·E, Midjourney, Stable Diffusion generate photorealistic images from text prompts.
🎮
Game AI
AlphaGo and AlphaStar used deep RL to beat world champions at Go and StarCraft II.
