Graph Representations

A graph G = (V, E) has vertices V and edges E. Before you can write algorithms, you need to choose how to store a graph in memory. The choice affects both space complexity and algorithm efficiency.

🗺️ Two Main Representations

Adjacency list: Each vertex stores a list of its neighbors. Good for sparse graphs. Space: O(V + E).

Adjacency matrix: A 2D array where matrix[i][j] = 1 if there's an edge. Good for dense graphs. Space: O(V²).

# Adjacency list (most common) graph = { 'A': ['B', 'C'], 'B': ['A', 'D', 'E'], 'C': ['A', 'F'], 'D': ['B'], 'E': ['B', 'F'], 'F': ['C', 'E'], } # Adjacency matrix for 4 nodes # matrix[i][j] = 1 means edge from i to j matrix = [ [0, 1, 1, 0], # A connects to B, C [1, 0, 0, 1], # B connects to A, D [1, 0, 0, 0], # C connects to A [0, 1, 0, 0], # D connects to B ]

Each vertex has an entry (V total) and each edge appears twice in undirected graphs (E total) → O(V + E).
Lesson 02 · CS Theory

Breadth-First Search (BFS)

BFS explores a graph level by level — all neighbors of the start, then their neighbors, etc. It uses a queue and finds the shortest path in unweighted graphs. Time: O(V + E).

🌊 BFS Algorithm

1. Start at source. Mark visited. Add to queue.
2. While queue not empty: dequeue a node, visit all unvisited neighbors, add them to queue.
3. The first time you reach a node, that's the shortest path.

from collections import deque def bfs(graph, start): visited = {start} queue = deque([start]) order = [] while queue: node = queue.popleft() order.append(node) for neighbor in graph[node]: if neighbor not in visited: visited.add(neighbor) queue.append(neighbor) return order # On graph above: bfs(graph, 'A') # → ['A', 'B', 'C', 'D', 'E', 'F'] (level order)

BFS Data Structure

BFS uses which data structure to track nodes to visit next? Type: queue or stack

BFS explores level-by-level (FIFO) — a queue. DFS explores depth-first — uses a stack (or recursion).
Lesson 03 · CS Theory

Depth-First Search (DFS)

DFS explores as deep as possible before backtracking. It uses a stack (or recursion) and is used for topological sort, cycle detection, connected components, and maze solving.

🔦 DFS Algorithm

1. Start at source. Mark visited.
2. For each unvisited neighbor: recursively DFS.
3. After all neighbors explored, backtrack.

Recursive DFS uses the call stack implicitly. Iterative DFS uses an explicit stack.

def dfs_recursive(graph, node, visited=None): if visited is None: visited = set() visited.add(node) print(node) for neighbor in graph[node]: if neighbor not in visited: dfs_recursive(graph, neighbor, visited) # Iterative DFS with explicit stack def dfs_iterative(graph, start): stack, visited = [start], {start} while stack: node = stack.pop() # LIFO print(node) for n in graph[node]: if n not in visited: visited.add(n) stack.append(n)

DFS vs BFS

You need to find the shortest path between two nodes in an unweighted graph. Which algorithm should you use? Type: BFS or DFS

BFS explores by level — the first time it reaches a node it has found the shortest path. DFS may find a longer path first.

Shortest Paths

Dijkstra's algorithm finds shortest paths from a source to all vertices in a weighted graph (non-negative weights). It's O((V + E) log V) with a min-heap priority queue — the algorithm behind GPS navigation.

🗺️ Dijkstra's Algorithm

1. Set all distances to infinity. Set source distance to 0.
2. Use a priority queue (min-heap by distance).
3. For each node: update distances to neighbors if shorter path found (relaxation).
4. Repeat until all nodes visited.

import heapq def dijkstra(graph, start): dist = {node: float('inf') for node in graph} dist[start] = 0 pq = [(0, start)] # (distance, node) while pq: d, u = heapq.heappop(pq) if d > dist[u]: continue # stale entry for v, weight in graph[u]: nd = dist[u] + weight if nd < dist[v]: dist[v] = nd heapq.heappush(pq, (nd, v)) return dist

Dijkstra Limitation

Dijkstra's algorithm fails with which type of edge weights? Type: negative

Dijkstra assumes once a node is visited, its shortest distance is finalized — this breaks with negative edges. Use Bellman-Ford instead.
